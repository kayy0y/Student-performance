# Student-performance
> Student Performance Analysis (Python) A data-driven project that analyzes and predicts student performance using Python. This system explores academic datasets to uncover insights, identify key factors influencing outcomes, and visualize performance trends effectively.

-> Features -Data cleaning and preprocessing -Exploratory Data Analysis (EDA) with visual insights -Performance trend charts (attendance, grades, study hours, etc.) -Correlation analysis to identify key learning factors -Predictive modeling using machine learning -Clear and interactive visualizations

-> Tech Stack -Python -Pandas, NumPy -Matplotlib / Plotly / Seaborn -Scikit-learn(for ML)
